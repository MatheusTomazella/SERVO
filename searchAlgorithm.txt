O sistema recebe uma string de busca e tem acesso a uma base com uma estrutura de tipo arvore.
Palavras chave da string são comparadas com as keyword de cada branch e o branch que mais se encaixa é o proximo. O processo é repetido.

Ex: abrir opções de som

{
    abrir {
        keywords [ abrir, abra, entrar, sla é mais abrir mesmo ]
        audio {
            keywords [ audio, som ]
            configurações {
                keywords [ configurações, configs, opções ]
                url
            }
        }
        jogo {
            keywords [ jogo ]
            lol {
                keywords [ lol, league of legends, ligui ofi legis ]
                url
            }
        }
    }
    pesquisar {
        keywords [ pesquisar, buscar ]

    }
}

path: abrir ( abrir ) -> audio ( som ) -> configurações ( opções ) -> url
